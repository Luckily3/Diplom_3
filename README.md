# Diplom_2
Diplom Unit Tests

## Технологии проекта:

Java 11 (Amazon Corretto 11), Maven, JUnit4, Selenium, RestAssured, Allure

## Как запускать:

Запуск тестов в Яндекс.Браузере:
`mvn test -Dbrowser=yandex`

Запуск тестов в  Google Chrome:
`mvn test -Dbrowser=chrome`

Запуск отчета Allure:
`mvn allure:serve`
